常用软件

idea常用快捷键

  • Command + O 查找类

  • Command + Delete 删除行

  • Command + B 方法/类引用

  • Command + Option + B 接口引用，直接进入实现

  • Command + Option + 向左/向右 回到光标之前/之后的点

  • Command + Option + L 格式化代码

  • Command + Shift + Enter 自动结束代码，行末自动添加分号

  • Command + P 显示方法参数

  • Command + Option + T 包围代码

  • Command + W 关闭当前选项卡

  • Command + N 快速生成代码（getter，setter，构造方法，toString）

  • Command + L 按行号查找+

  • Control + O 重写父类方法

  • Control + I 实现接口方法

  • Control + Command + F 最大化窗口+

  • Shift + Command + 上下方向键 上下移动代码快
