一、创建分支：

  1. git pull 拉取最新分支代码
  2. git branch 分支名 创建一个本地分支
  3. git checkout 分支名 切换到本地分支
  4. git push origin 分支名 推送分支代码到远端

二、删除分支：

  1. 先切换到其他分支
    git checkout master

  2. 删除本地分支
    git branch -d 分支名

  3. 如果删除不了可以强制删除
    git branch -D 分支名

  4. 删除远程分支
    git push origin –delete 分支名

三、提交代码

  1. git add . 先添加到本地，注意这里有个’.’,代表所有文件

  2. git commit -m “提交说明”

  3. git pull origin master 先pull一下，以免有代码覆盖等问题

  4. git push origin master 推送到远端
